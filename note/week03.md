## vite.dev
Vite (อ่านว่า "วีท" เป็นภาษาฝรั่งเศสแปลว่า "เร็ว") คือเครื่องมือที่ช่วยให้เหล่านักพัฒนาเว็บไซต์ (Frontend Developer) สามารถสร้างโปรเจกต์และรันโปรเจกต์เพื่อเขียนโค้ดได้รวดเร็วยิ่งขึ้นครับ มันถูกสร้างมาเพื่อแก้ปัญหาความล่าช้าของเครื่องมือรุ่นเก่าๆ อย่าง Webpack

หน้าที่หลักของ Vite คือเป็น Build Tool และ Development Server ครับ โดยปกติเวลาเราเขียนเว็บไซต์ด้วย Framework สมัยใหม่ (เช่น React, Vue, Svelte) เราไม่สามารถส่งไฟล์เหล่านั้นให้ Browser อ่านได้โดยตรง ต้องมีตัวกลางช่วยจัดการ ซึ่ง Vite ทำหน้าที่ดังนี้:

    Project Scaffolding: ช่วยสร้างโครงสร้างโปรเจกต์เริ่มต้นให้เราเลือกได้ว่าจะใช้ React, Vue หรือภาษาอื่นๆ

    Dev Server: สร้างเซิร์ฟเวอร์จำลองในเครื่องเรา เพื่อให้เราดูผลลัพธ์ของเว็บไซต์ที่กำลังเขียนได้แบบ Real-time

    Bundling: เมื่อเขียนเสร็จแล้ว Vite จะทำการ "มัดรวม" ไฟล์ทั้งหมด (JS, CSS, รูปภาพ) ให้เล็กลงและพร้อมสำหรับการนำไปขึ้น Server จริง (Production)

ประโยชน์เด่นของ Vite

ทำไมเหล่านักพัฒนาถึงเปลี่ยนจากเครื่องมือเดิมๆ มาใช้ Vite กันหมด? นี่คือเหตุผลครับ:
1. ความเร็วที่ต่างกันราวฟ้ากับเหว (Speed)

ในเครื่องมือรุ่นเก่า เวลาเราแก้ไขโค้ดหนึ่งบรรทัด ระบบต้องไล่ตรวจสอบไฟล์ทั้งโปรเจกต์ใหม่ ทำให้รอนาน แต่ Vite ใช้เทคโนโลยี Native ESM ซึ่งจะโหลดเฉพาะไฟล์ที่เราแก้ไขเท่านั้น

    Cold Start: เปิดโปรเจกต์ขึ้นมาพร้อมทำงานได้เกือบจะทันที

    Hot Module Replacement (HMR): เมื่อคุณกด Save โค้ด หน้าเว็บจะอัปเดตแทบจะในพริบตา

2. รองรับเทคโนโลยีสมัยใหม่ครบครัน

Vite มาพร้อมกับการตั้งค่าที่พร้อมใช้งานทันที (Out of the box) ไม่ต้องไปนั่ง Config ให้ปวดหัว:

    รองรับ TypeScript ในตัว

    รองรับ JSX / TSX

    จัดการ CSS / PostCSS / Sass ได้อย่างง่ายดาย

3. การจัดการ Asset ที่ฉลาด

Vite ช่วยจัดการรูปภาพหรือไฟล์ต่างๆ ให้เราอัตโนมัติ เช่น การทำ Image Optimization หรือการแยกไฟล์ (Code Splitting) เพื่อให้ผู้ใช้งานโหลดหน้าเว็บได้เร็วขึ้น
